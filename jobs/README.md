# To-Do List Application

This is a simple to-do list application that allows users to add, view, update, and delete tasks.

## Features
- Add tasks
- View tasks
- Update tasks
- Delete tasks

## Installation
1. Clone the repository: `git clone https://github.com/sophierlewis-code/jobs.git`
2. Navigate to the project directory.
3. Install dependencies using `npm install`. (If using Node.js)

## Usage
- Run the application: `node app.js` (If using Node.js)
